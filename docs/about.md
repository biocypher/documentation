# About the BioCypher ecosystem

The BioCypher ecosystem facilitates research in biology and medicine by
streamlining the scientific workflow. It provides a suite of tools that help
manage knowledge in various ways, from building knowledge graphs to querying
them with AI.

<div class="grid cards grid-2" markdown>

    ---

    Automated and tested workflows for common and tedious tasks.

-   :material-creative-commons:{ .lg .middle } __Reducing redundancy__
